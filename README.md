## Problem Statement

This Excel Dashboard provides a comprehensive view of business performance, enabling stakeholders to analyze key metrics such as **Revenue**, **Profit**, **Expenses**, **Category Performance**, and **Region-wise Insights**.

Using Pivot Tables, Charts, and Slicers, the dashboard transforms raw transactional data into actionable insights that help identify high-performing areas, low-performing segments, cost centers, and growth opportunities.

---

## Steps Followed

* **Step 1:** Loaded and reviewed the raw dataset containing transaction-level details.
* **Step 2:** Cleaned the data: formatted dates, removed duplicates, checked missing values, and organized categorical fields.
* **Step 3:** Created **5 Pivot Tables** for:

  * Revenue Analysis
  * Profit Analysis
  * Expense Breakdown
  * Category-Level Insights
  * Region-Wise Performance
* **Step 4:** Built Pivot Charts (Column, Bar, Line, Pie) and linked them to the Pivot Tables.
* **Step 5:** Added Slicers for Category, Region, Customer Segment, Department, and Product Line for interactive filtering.
* **Step 6:** Applied number formats (comma separators, currency formats).
* **Step 7:** Designed KPI cards for Revenue, Profit, and Expenses.
* **Step 8:** Arranged all visuals on a dedicated Dashboard sheet using shapes, icons, and consistent formatting.
* **Step 9:** Synced slicers across all Pivot Tables for unified filtering.
* **Step 10:** Verified chart connections, slicer interactions, and calculation accuracy.
